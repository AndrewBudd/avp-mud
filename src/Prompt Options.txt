The Following Prompt Flags are currently available:
 %A = Armor Value (Partially complete)
 %g = Current Credits (On Person)
 %h = Current Hitpoints (Integer)
 %H = Maximum Hitpoints (Integer)
 %i = Current Invisibilty Status
  (Mortal: Only shows when invisible, Displays '(Invis) ')
  (Immortal: Only shows when wizinvis, Displays '(Invis #) ')
 %I = Current Invisibilty Level (Immortal Only --- '104')
 %m = Current Force Points (Integer)
 %M = Maximum Force Points (Integer)
 %r = Current Room Vnum (Immortal Only)
 %R = Current Room Vnum (Immortal Only --- Requires CONFIG +VNUM)
 %u = Current Users (Integer)
 %U = Max Ever Players (Integer)
 %v = Current Movement Points (Integer)
 %V = Maximum Movement Points (Integer)
